Télécharger VALAPP Desktop

Si Windows affiche un avertissement "SmartScreen", cliquez Informations complémentaires → Exécuter quand même. Ce message apparaît pour tout nouveau logiciel non certifié.

Installation en 3 étapes

1

Télécharger

Cliquez sur le bouton ci-dessus pour télécharger l'installateur depuis GitHub Releases.

2

Installer

Lancez le fichier .exe, acceptez l'avertissement SmartScreen si nécessaire, et laissez l'installation se terminer.

3

Connecter

Ouvrez ValAPP, connectez votre compte Riot via la fenêtre de login officielle — et c'est tout.
